# 本地化术语表

Cosmosh 的术语表是跨语言矩阵，不在中文文档中重复维护。请以英文源文档中的同一张术语表为事实来源：

[Localization Terminology](../../../developer/design/localization-terminology.md)

## 使用规则

- 命名产品区域、功能模块、设置分类和主要 SSH/SFTP 概念时，先查阅英文源文档中的术语表。
- 如果需要新增或调整术语，只修改英文源文档中的矩阵，并在本页保持指向该源文档。
- 不要在本页复制术语表内容，避免 `zh-CN`、`en` 或未来更多语言出现漂移。
- 这份术语表不是完整的 locale key 索引；不要加入每个字段标签、按钮、状态、校验消息或设置项。
